# Gemini Flask Chatbot

A full-stack AI chatbot web application built using Flask, HTML, JavaScript, and Google Gemini API.

## Live Demo

https://gemini-flask-chatbot-2wjm.onrender.com

## Features

- Real-time AI chat responses
- Flask backend integration
- Gemini API integration
- Interactive frontend UI
- Secure API key handling using `.env`

## Tech Stack

- Python
- Flask
- HTML
- JavaScript
- Google Gemini API

## Run Locally

```bash
pip install -r requirements.txt
python app.py
